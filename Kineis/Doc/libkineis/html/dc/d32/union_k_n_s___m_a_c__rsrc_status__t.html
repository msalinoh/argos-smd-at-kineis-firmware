Detailed Description

Bitmap listing ressources currently used by MAC protocol (timers, RTC, RAM, ...)

By comparing this list of active peripherals and the corresponding HW modules (cf corresponding MCU wrappers), it is possible to deduce what can be the deepest LPM without impacting the MAC protocol.
