62 /**
63  * @brief Disable all GPIO as not used for the moment
64  *
65  * @todo Use this fct for LPM optimization (especially when LPM standby)
66  *
67  * This function disables GPIOs by:
68  * * clocking all ports
69  * * Configure all GPIO port pins in Analog Input mode (floating input trigger OFF)
70  * * unclocking all ports
71  *
72  * It is done in a way to save some current drain while GPIOs are not needed.
73  *
74  * So far, GPIOs are mainly connected to SX1255 transceiver. Thus it is needed when powering ON,
75  * TX, RX, SAT detection and so on. transceiver driver will need to set GPIOs to correct state
76  * when needed.
77  *
78  * \note This GPIO could be disabled back at end of RF any action (RX/TX/SAT_DETECT). So far,
79  * it will only go back disabled when entering LPM shutdown mode.
80  *
81  * \note GPIOs used by peripherals, such as SPI, SWD.JTAG, LPTIM are not disabled here.
82  *
83  * \todo Disable SWD GPIOs (used for ST-Link debugging) in release build. Note, this will:
84  * * disable SWD attach/debug on a running build
85  * * but it remains possible to reflash by pressing RESET pin/button of the uC at the start
86  * of flashing procedure
87  */
